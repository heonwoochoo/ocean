## ✨ Motivation
지구온난화와 무분별한 쓰레기 배출로 인한 환경오염은 항상 사회적 문제 거론되고 있습니다. 
</br>
그중 가장 영향력을 많이 받는 곳은 극지방이 아닐까 싶습니다. 
</br>
제가 생각하는 미래의 남극과 북극의 모습을 webgl을 활용하여 구현했습니다.
